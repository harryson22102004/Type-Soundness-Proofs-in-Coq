# Type Soundness Proofs Simulation
Exploring programming language theory and the formal verification of type systems.

### 🔬 Research Highlights
- **Progress & Preservation**: Simulates the two pillars of type soundness (Progress: A well-typed term is not stuck; Preservation: Evaluation preserves types).
- **Lambda Calculus**: Models a typed lambda calculus system to verify property consistency.
- **PL Theory Focus**: Demonstrates foundation in the "Correctness by Construction" philosophy.

**Tech Stack:** Python.
